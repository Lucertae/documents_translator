# Privacy Policy - LAC TRANSLATE

**Ultimo aggiornamento**: Gennaio 2024

## Introduzione

LAC Software ("noi", "nostro", "la Società") rispetta la privacy degli utenti di LAC TRANSLATE. Questa Privacy Policy descrive come raccogliamo, utilizziamo e proteggiamo le informazioni quando utilizzi il nostro software.

## Principio Fondamentale

**LAC TRANSLATE funziona completamente offline e locale. I tuoi dati rimangono sul tuo computer e non vengono inviati a server esterni senza il tuo consenso esplicito.**

## Raccolta Dati

### Dati Raccolti Localmente

Il software può raccogliere e memorizzare localmente sul tuo computer:

- **File PDF**: I documenti che traduci rimangono sul tuo computer
- **Traduzioni**: Le traduzioni generate sono salvate localmente
- **Impostazioni**: Configurazioni e preferenze utente
- **Log**: File di log per diagnostica (solo sul tuo computer)
- **Licenza**: Informazioni sulla licenza (chiave seriale, ID hardware)

### Dati NON Raccolti

Non raccogliamo:
- Informazioni personali identificative (nome, email, ecc.) - salvo quelle fornite volontariamente per supporto
- Contenuti dei documenti tradotti
- Dati di navigazione o utilizzo del software
- Informazioni su altri software installati

## Utilizzo Dati

### Uso Locale

I dati raccolti localmente sono utilizzati esclusivamente per:
- Funzionamento del software
- Salvataggio preferenze utente
- Diagnostica errori (solo localmente)

### Servizi Online (Opzionali)

Se utilizzi **Google Translate** (online):
- I testi da tradurre vengono inviati a Google
- Google ha la propria Privacy Policy che si applica
- Non abbiamo controllo sui dati inviati a Google
- Per privacy totale, usa **Argos Translate** (offline)

Se utilizzi **Argos Translate**:
- Tutto avviene offline sul tuo computer
- Nessun dato viene inviato a server esterni

## Condivisione Dati

**Non condividiamo i tuoi dati con terze parti**, eccetto:
- Servizi di traduzione online (solo se scegli Google Translate)
- Servizi di supporto tecnico (solo informazioni necessarie per assistenza)

## Sicurezza Dati

- **Crittografia**: I file di licenza utilizzano hash sicuri
- **Accesso Locale**: Solo utente con accesso al computer può vedere i dati
- **Nessun Server**: Non utilizziamo server centralizzati per memorizzare dati utente

## Diritti Utente

Hai il diritto di:
- Accedere ai tuoi dati (sono tutti sul tuo computer)
- Eliminare i tuoi dati (disinstalla il software o elimina directory config/logs/output)
- Non utilizzare servizi online (usa solo Argos Translate)
- Contattarci per domande sulla privacy

## Supporto Tecnico

Se contatti il supporto tecnico, potremmo chiederti:
- Informazioni sistema operativo
- Messaggi di errore
- ID Hardware (per problemi licenza)
- Versione software

**Non condividere mai il contenuto dei documenti che traduci a meno che non sia strettamente necessario per il supporto.**

## Modifiche Privacy Policy

Ci riserviamo il diritto di modificare questa Privacy Policy. Le modifiche saranno comunicate tramite aggiornamenti software o email (se fornita).

## Conformità GDPR

Se sei residente nell'UE, questa Privacy Policy è conforme al GDPR (General Data Protection Regulation):
- **Consenso**: Utilizzando il software acconsenti a questa policy
- **Diritto all'oblio**: Puoi eliminare tutti i dati disinstallando il software
- **Portabilità**: Tutti i dati sono locali sul tuo computer
- **Trasparenza**: Questa policy descrive chiaramente l'uso dei dati

## Conclusione

La privacy è fondamentale per LAC TRANSLATE. Per massima privacy:
1. Usa Argos Translate (offline) invece di Google Translate
2. Mantieni i file PDF locali sul tuo computer
3. Non condividere informazioni sensibili con il supporto a meno che non sia necessario
